Completely redesign a high-end personal portfolio website for a freelance web designer named Evans Bilong.

This is not a standard portfolio. The goal is to create a bold, experimental, visually striking experience inspired by high-end editorial design, brutalist layouts, and Awwwards-level portfolios.

---

ART DIRECTION (CRITICAL)

The design must follow a strong visual direction:

- Dominant deep black background
- Red is used as LIGHT and GLOW, not as flat UI color
- High contrast, cinematic atmosphere
- Inspired by luxury, fashion, and experimental digital studios

Avoid generic UI design at all costs.

---

VISUAL STYLE

- Use large, bold, oversized typography (impact-first design)
- Create tension using asymmetry and broken layouts
- Use strong visual imagery (dark photography, high contrast, dramatic lighting)
- Avoid clean grid-based layouts — introduce controlled chaos

---

HERO SECTION (EXPERIENCE)

Design a powerful fullscreen hero:

- Include a strong visual (distortion, slicing, glitch, or layered effect)
- Minimal text:
  - "Evans Bilong"
  - "Freelance Web Designer"
- Add a subtle motion concept (glitch, distortion, or parallax)
- The hero should feel like a digital art piece, not a standard header

---

PROJECTS SECTION (IMMERSIVE)

- Use a horizontal or scroll-driven project experience
- Projects should feel like editorial spreads, not cards
- Each project:
  - Large image
  - Minimal text
  - Strong composition
- Include experimental transitions (scale, reveal, distortion)

---

TYPOGRAPHY

- Use bold, modern sans-serif fonts
- Mix large headlines with very small supporting text
- Play with spacing, line breaks, and alignment
- Typography should feel like a core visual element, not just text

---

COLOR USAGE

- Black is dominant
- Red appears as:
  - glow
  - gradients
  - lighting effects
  - subtle highlights
- Avoid flat red blocks unless used intentionally for impact

---

MICRO-INTERACTIONS

- Add subtle pulsing circular elements (radar-like effect)
- Use smooth transitions and hover interactions
- Include glitch or distortion effects in small doses
- Everything should feel alive but controlled

---

PERSONAL IDENTITY

- Include a portrait section for Evans Bilong
- The image should feel integrated in the design:
  - dark
  - cinematic
  - possibly blended with background
- Use placeholder image for now

---

UX RULES

- Despite the experimental design, navigation must remain clear
- Keep key sections:
  - Hero
  - Projects
  - About
  - Contact
- Maintain usability and readability

---

AVOID

- Generic portfolio layouts
- Clean grid-only design
- Basic UI components
- Overused templates
- Flat, boring compositions

---

GOAL

The final result should feel:
- premium
- bold
- experimental
- memorable

This should look like a portfolio that wins design awards, not a junior designer template.
