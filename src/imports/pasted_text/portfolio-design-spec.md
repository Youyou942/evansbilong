Design a high-end personal portfolio website for Evans Bilong, a freelance web designer.

The goal is to create a bold, experimental, and premium experience inspired by fashion, editorial design, and Awwwards-level portfolios.

This must NOT look like a generic template.

---

ART DIRECTION

- Use a deep black background (#050505)
- Use a strong red (#FF0000) as accent color
- Red should feel like light or glow, not flat UI color
- Add subtle grain/noise texture for a cinematic and gritty atmosphere

---

LAYOUT

- Use an asymmetric and broken grid layout
- Avoid perfect alignment and standard sections
- Elements can overlap and create visual tension
- Mix very large elements with very small details

---

HERO SECTION

- Fullscreen immersive hero
- Strong visual (dark image, high contrast, cinematic)
- Add a distortion or layered effect (glitch / slicing feel)
- Minimal text:
  - Evans Bilong
  - Freelance Web Designer

---

TYPOGRAPHY

- Use bold, oversized titles (very large scale)
- Combine:
  - modern sans-serif for headlines
  - monospaced font for small UI text and labels
- Play with contrast in size and spacing

---

PROJECTS SECTION

- Use an immersive layout (horizontal scroll or large sections)
- Projects should feel like visual compositions, not simple cards
- Large images, minimal text
- Add hover effects (zoom, distortion, or reveal)

---

UI ELEMENTS

- Add a small “radar-like” pulsing circular element
- Add thin lines or markers to create a technical / scanning feel
- Use a custom cursor with a small red dot that reacts on hover

---

INTERACTIONS

- Smooth and slightly heavy transitions
- Subtle pulse effect on important elements
- Scroll-based reveals (not basic fade-ins)
- Small glitch or distortion effects used sparingly

---

PERSONAL SECTION

- Add a portrait section (placeholder image for now)
- The image should feel integrated into the design
- Dark, cinematic style

---

UX

- Keep navigation simple and clear
- Avoid clutter
- Balance experimental design with usability

---

AVOID

- generic layouts
- standard UI blocks
- basic grids
- overused portfolio patterns

---

GOAL

The result must feel:

- bold
- premium
- experimental
- visually striking

This should look like a portfolio that stands out immediately and feels high-end.
